将 XiJi 文件夹放入服务器中
http://localhost/XiJi/

1、登录，把鼠标移到登录注册上，在滑出的框中，点击登录。(账号:15852377374 密码：12345678)
2、注册，把鼠标移到登录注册上，在滑出的框中，点击立即注册。(账号可自行注册,用的是千锋接口)
3、购物车，把鼠标移到购物车上，直接点击购物车。
4、商品列表页面是"精品家居馆",点击任意一个商品，跳转到对应的商品详情页，
5、在商品详情页点击加入购物车，跳转到购物车页面，
6、在商品详情页不点击加入购物车，购物车中是没有的。
7、如需再选商品，再点击导航栏上的精品家居馆
8、所有页面中其他效果，可自行浏览。
